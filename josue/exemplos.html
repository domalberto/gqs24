Proteção Contra XSS

Exemplo de proteção contra Cross-Site Scripting (XSS) usando a função encodeURIComponent para evitar injeção de scripts maliciosos.


                // Proteção contra XSS
                function sanitizeInput(input) {
                    return encodeURIComponent(input);
                }

                const userInput = '';
                const safeInput = sanitizeInput(userInput);
                console.log(safeInput); // %3Cscript%3Ealert(%22XSS%22)%3C/script%3E
